## Slide 2: Project Overview
### What is Eventomania?
- **Comprehensive Event Management Platform** for educational institutions
- **Streamlined workflow** for event creation, approval, and management
- **Multi-role system** supporting different user types
- **Modern web application** built with React and Node.js

### Key Benefits:
- ✅ Centralized event management
- ✅ Automated approval workflows
- ✅ Digital certificate generation
- ✅ Email notifications
- ✅ Photo and report management

## Slide 3: Problem Statement
### Challenges in Traditional Event Management:
- **Manual processes** leading to inefficiencies
- **Lack of centralized system** for event tracking
- **Paper-based approvals** causing delays
- **Difficulty in managing** multiple committees
- **No automated notifications** for stakeholders
- **Manual certificate generation** and distribution

### Our Solution:
Eventomania provides a **digital platform** that automates and streamlines the entire event management lifecycle.

## Slide 4: System Architecture
### Technology Stack

**Frontend:**
- React.js 18.3.1
- Material-UI (MUI) for UI components
- Redux Toolkit for state management
- React Router for navigation
- Formik & Yup for form validation

**Backend:**
- Node.js with Express.js
- MongoDB with Mongoose ODM
- JWT for authentication
- Multer for file uploads
- Nodemailer for email services

**Additional Features:**
- PDF generation for certificates
- Image compression with Sharp
- CORS and security middleware

## Slide 5: User Roles & Authentication
### Three-Tier User System

**1. Admin (Super Admin)**
- Overall system management
- Event approval authority
- Committee and user management
- System-wide analytics

**2. Convenor (Committee Head)**
- Committee-specific event management
- Member management
- Event creation and approval
- Committee dashboard access

**3. Member (Committee Member)**
- Event creation capabilities
- Photo and report uploads
- Limited administrative access

**4. Students/Faculty (Event Participants)**
- Event registration
- Certificate download
- Event information access
